Inline layout - vertical in horizontal

In following document, inline layout is wrote as

<layout width="380" height="200" direction="vertical" align="right"> ～ </layout>

Notice that all of the text must be filled in the inline layout space.
If there is some contents that overflow the inline layout, nehan will just ignore it.
